# Tujuan Repository
Repository ini berisi projek-projek activity dari matakuliah Rekayasa Perangkat Lunak 2 (RPL2).

## Isi Repository
Dalam Repository ada 6 folder/projek yang di kerjakan mulai dari minggu 1-6 dalam satu repository.

### Project M1
Berisi kodingan Pengenalan Rekayasa Perangkat Lunak

### Project M2
Berisi contoh Object Oriented Programing (OOP)

### Project M3
Berisi Konsep Model View Controller (MCV)

### Project M4
Berisi KONSEP DASAR OBJECT RELATIONAL MAPPING (ORM) DAN FRAMEWORK HIBERNATE

### Project M5
Berisi KONSEP FRAMEWORK SPRING, PEMBUATAN PROJECT SPRING DAN HIBERNATE

### Project M6
Berisi Implementasi Aop Dan Dependency Injection Pada Project Spring Dan Hibernate
